5 TAG DESCONTINUADAS

  • <applet> : Insere um applet na página - Foi substituido por <object>
  • <isindex> : Adiciona campo de pesquisa - Foi substituido por <form>
  • <dir> : Lista de diretórios - Foi substituido por <ul>
  • <strike> e <s> : Tacha o texto envolvo - Foi substituido por CSS
  • <acronym> : Cria uma abreviação com dica do que significa - Foi substituido por <abbr>
  • <dir> : Lista de diretórios - Foi substituido por <ul>
